# 5. 승인 필요 사항 요약

이 계획은 **사람 승인 없이 자동으로 실행되지 않습니다.** 아래 항목은 담당자의 명시적 확인·승인이 필요합니다.

1. 예산 530만원 및 예비비 30만원 증액 — 정식 결재
2. 실제 대관 장소 확정 (현재는 데모 예시)
3. 참석자 알레르기·식이 제한 명단 공유 (개인정보)
4. 보험 가입 및 증권 확인
5. 안전 담당자 응급처치 자격 확인
6. 화재·대피 동선 최종 확인 (실제 장소 기준)
