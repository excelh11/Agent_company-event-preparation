# 이번 행사 정보 (Event Params)

탐색 단계 산출물. 이 값을 기준으로 prep-coordinator와 scheduler가 계획 단계 산출물을 만든다.

| 항목 | 값 | 비고 |
| --- | --- | --- |
| 행사 유형 | 팀/회사 오프사이트 워크숍 | |
| 날짜 | 2026-10-16 (금) | "10월 셋째 주 금요일" 기준 계산값 — 실제 날짜 재확인 권장 |
| 인원 | 20명 | |
| 예산 | 총 500만원 (1인당 약 25만원) | 확인 필요: 교통비 포함 여부 |
| 장소 형태 | 야외 중심, 우천 대안 필수 | 사용자 확인 완료(2026-07-10) |
| 구체적 장소 | (데모 예시) 경기도 가평 소재 야외 잔디마당 보유 리조트 — 실내 다목적홀 우천 대안 완비 | **실제 대관 여부 미검증(예시 장소)**. 실제 행사라면 대관 가능 여부·연락처를 직접 확인해야 함 |
| 행사 목적 | 확인 필요 | 사용자가 명시하지 않음 — 팀 오프사이트/친목·워크숍 성격으로 가정 |

## 확인 필요 요약

- 실제 날짜 재확인 (달력 기준 계산값 사용 중)
- **장소는 데모용 예시로 지정됨** — 실제 행사라면 반드시 실재 대관 가능 여부·주소·연락처를 확인해야 함
- 예산에 교통비 포함 여부
- 행사 목적(친목/워크숍/전략회의 등)이 준비물 구성에 영향을 줄 수 있음
