# 회사 행사 표준 규칙 (Persistent Instruction)

이 파일은 행사마다 새로 만들지 않고 계속 재사용하는 회사 표준 규칙이다. 이번 행사에만 해당하는 값(날짜, 예산 금액, 인원)은 여기 두지 않고 `artifacts/00-event-params.md`에 둔다.

최초 실행 전에 아래 항목을 회사 실제 정책에 맞게 채워야 한다. 채우지 않은 항목은 Orchestrator가 첫 실행 시 사용자에게 확인 질문으로 던진다.

## 예산 승인 절차

- 예산 상한: 확인 필요 (예: 1인당 상한액, 총액 상한)
- 상한 초과 시 승인권자: 확인 필요
- 법인카드/증빙 규정: 확인 필요

## 안전 필수 확인 항목 (모든 행사 공통)

- 응급 연락처(행사 담당자, 인근 병원) 사전 확보
- 참석자 알레르기·식이 제한 사전 조사
- 실외 요소가 있으면 우천/혹서 등 기상 대안 계획
- 이동 수단이 있으면 안전벨트/음주운전 금지 고지
- 화재·대피 동선 확인(실내 행사, 대관 장소 기준)
- 확인 필요: 회사 자체 필수 안전 항목이 더 있다면 추가

## 복장·행동 규정

- 확인 필요 (예: 세미 캐주얼, 사내 행동강령 준수 안내 문구)

## 개인정보 처리 원칙

- 참석자 명단, 알레르기 등 개인정보는 행사 준비 목적 외 공유 금지
- 명단 발송 전 사람 승인 필수 (Orchestrator 승인 게이트에서 처리)

## 문서 형식

- 확인 필요 (예: 사내 공지 템플릿, 결재 문서 양식이 있다면 경로 기재)
